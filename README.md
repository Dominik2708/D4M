## 1. Projektüberblick

**Was ist das Projekt?**  
HAITI ist ein Schülerunternehmen der Klasse HA25B an der BBS I Lüneburg (einjährige Berufsfachschule Wirtschaft, Schwerpunkte Handel & IT). Das Unternehmen entwickelt, produziert und vermarktet nachhaltige Produkte mithilfe moderner 3D-Druck-Technologie und in Kooperation mit der BBS II (Holzwerkstatt).

**Rechtlicher Status:** Nicht-kommerziell, von der Mehrwertsteuer befreit, unterliegt besonderen Haftungsregeln für Schülerunternehmen (§ 5 TMG). Schulleiter: Heiko Lüdemann (Oberstudiendirektor).

**Schule:** BBS I – Berufsbildende Schulen, Spillbrunnenweg 1, 21337 Lüneburg  
**Kontakt:** schuelerunternehmen@bbs1-lueneburg.de · 04131 – 207 400

## 3. Kommende Termine

### Wirtschaft-Live-Messe – 18. März 2026
- **Ort:** Berufsschule Niedersachsen (landesweit)
- **Art:** Wettbewerb mit ~20 teilnehmenden Schülerunternehmen
- **Wettbewerbskategorien:**
  - Bester Messestand
  - Bester Messebeitrag
  - Bester Imagefilm
  - Beste Fotopräsentation der Geschäftsidee

### Summer-Sale – Juni 2026
- **Ort:** BBS I Lüneburg (Ende des Schuljahres)
- **Inhalt:** Restposten zu reduzierten Preisen + neu entwickelte Sommerartikel (z. B. 3D-gedruckte Insektenschutzlösungen für Getränke)

## 5. Quiz – Wasser-Plastik-Quiz

Das Quiz auf der Website testet Nutzerwissen zu Plastikverschmutzung und Weltmeeren. Es besteht aus **18 Fragen** mit je 4 Antwortoptionen. Zwei zufällige Fragen geben **doppelte Punkte**.

| # | Frage | Richtige Antwort |
|---|---|---|
| 1 | Wie lange dauert es, bis sich Plastikmüll im Meer abgebaut hat? | 500 Jahre |
| 2 | Wie viele Tiere sterben jährlich am Plastikmüll im Meer? | 1,5 Millionen |
| 3 | Welches ist das größte Korallenriff auf der Erde? | Great Barrier Reef |
| 4 | Wie lange ist das größte Korallenriff? | 2.300 km |
| 5 | Wie tief ist der Marianengraben an seiner tiefsten Stelle? | 11 km (≈ 11.000 m) |
| 6 | Wie tief kann Licht ins Meer eindringen (für Menschen sichtbar)? | 300 m |
| 7 | Wie viel Prozent der Ozeane sind bereits erforscht? | 5–10 % |
| 8 | Wie lang ist die längste Bergkette im Meer? | 65.000 km |
| 9 | Welcher Ozean ist der größte der Erde? | Pazifischer Ozean |
| 10 | Wie viel Prozent der Erdoberfläche sind mit Wasser bedeckt? | 71 % |
| 11 | Welches Meer hat den höchsten Salzgehalt? | Totes Meer |
| 12 | Wie heißen die weltweiten warmen Meeresströmungen? | Thermohaline Zirkulation |
| 13 | Welche Farbe des Lichts dringt am tiefsten ins Meer? | Blau |
| 14 | Welches Meer ist das größte Binnenmeer der Welt? | Kaspisches Meer |
| 15 | Wie nennt man massenhaftes Algenwachstum, das das Wasser grün färbt? | Algenblüte |
| 16 | Wie nennt man die Zone, in der das Licht vollständig verschwindet? | Aphotische Zone |
| 17 | Was ist die Hauptursache für Ebbe und Flut? | Schwerkraft von Mond und Sonne |
| 18 | Wie viel Prozent des Meeresmülls besteht aus Plastik? | 80 % |

## 7. Die Website

Die Website ist eine einseitige Anwendung (Single Page Application) mit scrollbasierter Navigation. Sie ist vollständig **zweisprachig (Deutsch/Englisch)** und unterstützt einen **Hochkontrast-Modus** für bessere Barrierefreiheit.

### Seitenaufbau (in Reihenfolge)

| Abschnitt | Inhalt |
|---|---|
| Header | Navigation, Sprachumschalter (DE/EN), Hochkontrast-Toggle |
| Hero | Hauptbild, Projektbeschreibung, rechtlicher Hinweis, CTA-Buttons |
| Produkte | Produktkategorien mit Bildergalerien und Feature-Listen |
| Quiz | 18 Fragen, zufällig gemischte Antworten, Doppelpunkt-Runden |
| Über uns | Teambeschreibung, Methodik, Kooperationspartner |
| Termine | Kommende Veranstaltungen mit Detailkarten |
| Archiv | Vergangene Projekte nach Jahren gruppiert |
| Kontakt | Kontaktformular, Kontaktdaten, eingebettete Karte, Anfahrt |
| Footer | Impressum, Datenschutz, Copyright |

### Tech Stack

| Bereich | Technologie |
|---|---|
| Framework | Next.js 15 (App Router, Turbopack) |
| UI-Bibliothek | React 19 |
| Sprache | TypeScript 5 |
| Styling | Tailwind CSS 4 |
| UI-Komponenten | Radix UI |
| Icons | Lucide React |
| Internationalisierung | i18next + react-i18next |
| E-Mail (Kontaktformular) | EmailJS (kein Backend nötig) |

### Lokale Entwicklung

```bash
# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev
```

Die App läuft dann unter [http://localhost:3000/D4M](http://localhost:3000/D4M).

## 8. Arbeitsweise & Teamregeln

**Sprints:** Retrospektive, Planung, Pausen

**Definition of Done:**
- Aufgabe erledigt + getestet
